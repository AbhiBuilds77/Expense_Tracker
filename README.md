# 💰 Expense Tracker

A simple and responsive Expense Tracker built with **React** and **Tailwind CSS**. It allows users to add expenses, view a list of all expenses, calculate the total amount spent, and switch between multiple UI themes.

---

## 🚀 Features

- ➕ Add new expenses
- 📝 Enter expense description and amount
- 📋 Display all added expenses
- 💵 Automatically calculate total expenses
- 🎨 Multiple themes (Light, Dark, Pastel)
- 📱 Responsive design
- ⚡ Built using React Hooks

---

## 🛠️ Tech Stack

- React
- Tailwind CSS
- JavaScript (ES6+)
- Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/expense-tracker.git
```

Navigate to the project folder

```bash
cd expense-tracker
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

---

## 📖 Usage

1. Enter the expense amount.
2. Enter a short description.
3. Click **Add**.
4. The expense appears in the list.
5. The total expense is updated automatically.
6. Switch between Light, Dark, and Pastel themes using the theme buttons.
